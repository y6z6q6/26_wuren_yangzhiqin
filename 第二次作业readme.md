# 作业记录
第二次的作业。作业一，我学习了Astar算法，了解其与宽搜的不同之处：F启发函数。由代价函数指导程序每次从最小代价点出发寻找
结果。作业二，是学习线性回归，我本来是想手写矩阵计算的，但是后面通过ai了解到python里面有专门的关于矩阵的库，方便计算。
其中我选择的几个参数分别是建造年份 地块面积 卧室数量 厕所数量 室内面积 车库数量 学校评分，但是我取了50个房价进行线性回归，
拟合出来的结果偏差有点大，我觉得有俩个原因，第一个可能是我选择的这几个参数对房价的影响不是很大，第二个可能是我选择的样本数量过少。改进一下这两个方面，我觉得拟合出来的数据会接近正确数据。作业三，是调用SVM库来分类，我没想到这个库里面自带花的数据，学会调用
现成的函数对能提高写代码的效率，前提是要了解其原理。作业四，使用KMEANS算法分类，我认为这个算法处理数据的方法很巧妙，通过随机
指认中心点，然后把每个点分配给最近的中心点，再进行检验，但是我在实际调试中发现，这个随机选中心点可能存在将两个中心点选在一个簇中的情况，
我觉得这可能需要改变中心点检验规则才能解决，在学习和编写KMEANS算法的过程中，我体会到了这个算法的神奇。
