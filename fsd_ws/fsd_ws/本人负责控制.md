# readme

**控制**

我们采用的是纯跟踪算法来控制小车实际行走路径，主要由3种函数组成：

订阅函数：
on_path()
        self.path = [
            (pose.pose.position.x, pose.pose.position.y)
            for pose in msg.poses
        ]
作用：接收规划好的路径，存入self.path，可以供纯跟踪算法计算偏向角和距离。

on_odom()
        self.state = (
            msg.pose.pose.position.x,
            msg.pose.pose.position.y,
            quaternion_to_yaw(msg.pose.pose.orientation),
        )
作用：读取车辆的当前世界坐标，计算出车辆当前的航向角yaw。实施输入。

处理函数：
publish_stop()
作用：发布停止指令，让车辆停下。

nearest_forward_index()
            math.atan2(dy, dx)

作用：寻找车辆前方最近的路径点，也就是纯跟踪算法中的预瞄点。为了严格使预瞄点在车辆前方，我们加入了限制条件：
if local_x < 0.0:continue，这样不会让小车走回头路和转圈。

path_heading()       
            if index + 1 >= len(self.path):
            index = max(0, index - 1)
倒数第二个点就直接看成最后一个点
        if index + 1 >= len(self.path):
            return 0.0
最后一个点就直接返回0，角度为0
作用：算出小车质心与预瞄点的角度，用来减去车身航向角，得到α

upcoming_bend()
        for i in range(start_index + 1, limit + 1):
            max_delta = max(max_delta, abs(normalize_angle(self.path_heading(i) - base)))
        return max_delta
遍历未来18个点，找到最大角度
作用：计算每两个路径点之间的转角，取最大值返回，用来了解未来路径的弯曲程度，用来限制车辆最高和最低速度

核心函数：
on_timer()
作用：每50ms更新一次，根据判断前方弯曲程度，自动计算预瞄距离（越弯曲预瞄距离越短），然后根据预瞄距离找到最合适的路径点，
接着换算车辆坐标系，用纯跟踪算法计算前轮转角。
其中有两个优化点：
第一个：heading_index = min(
            len(self.path) - 2,
            target_index + max(1, int(2 + 3 * bend_scale)),
        )
        path_yaw = self.path_heading(heading_index)
        heading_error = normalize_angle(path_yaw - yaw)
用来判断未来路径是否弯曲，如果弯曲，则会提前让车辆转弯，转角为path_yaw - yaw。

第二个： turn_slowdown = min(1.0, abs(curvature) / 0.28)
        speed = self.target_speed * (1.0 - self.turn_slowdown_factor * turn_slowdown)
        speed = max(self.min_speed, speed)
根据当前预瞄点与车身角度（turn_slowdown）和未来路径弯曲程度（self.turn_slowdown_factor）来适当降低车辆速度。但是不能低于最低速度。


**总联调**

1、模块订阅
