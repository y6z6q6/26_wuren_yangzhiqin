# 作业记录
第三次实习是学习机器的深度学习。在给出的示例代码中，修改了判断的位置，我认为在每一次的训练中进行泛化检测会比较浪费时间，当然在我们这种数据量小的情况
下可能不会太久，但是在数据量变成几万至几十万数量级的时候时间成本会很高，所以我把它已到了epoch循环下，即每训练一次结束后来判断模型是否变优。其次，我把200次的初始给定训练次数削减到了80次，因为我发现训练200次很容易发生过拟合的情况，就是在train文件夹里的数据都会从94%正确率降低到88% 89%正确率。在调整
数据的同时，我也学习了程序读取文件和保存文件的知识。不知道是不是我电脑用户名是中文的原因，程序自带的保存在./train路径下的代码执行无效。我一开始是改成了C:\Users\杨智钦\Desktop\DL路径，但是后面我提交的时候想到别人要运行我的程序，data所在路径就不是这个了，所以在豆包的辅助下，我最终采用了相对路径的写法 BASE_DIR = os.path.dirname(os.path.abspath(__file__)) data_root = os.path.join(BASE_DIR, "dataset", "train")就是读取当前的文件夹路径，然后再把文件夹绝对路径和我要取数据的相对路径结合在一起，这样不管是再哪个电脑上运行，只要dataset文件夹的位置和代码文件的位置正确，就能加载数据。
